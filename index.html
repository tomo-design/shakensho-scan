      AIメカ君に相談機能（無料・任意）

      Google AI Studio で無料発行できる Gemini APIキーを設定すると、診断タブで「メカ君に相談」でき、ダイアグ・問診の内容をメカ君に投げて臨機応変な診断アドバイスを受けられます（クレジットカード登録不要。無料枠: 1日あたりの回数制限あり）。

      取得手順: aistudio.google.com/apikey ↗ → Googleアカウントでログイン → 「APIキーを作成」→ コピーして下に貼り付け

      どちらもAIが回答前に内部で「思考」してから答えます。高精度モードは複雑な複合症状向け。無料枠の上限に達した場合は自動で標準モデルに切り替えて再試行します。
      「無料枠の上限」をなくしたい場合（任意・従量課金）
      いまのAPIキーは無料で使えますが、1分あたり・1日あたりの回数制限があります。同じキーのまま、キーに紐づくプロジェクトの「課金」を有効にすると上限が大幅に上がります（有料・従量課金）。
      手順:
      ① Google AI Studio のAPIキー画面 ↗ を開く
      ② 使っているキーの行で Plan を確認 →「Set up Billing（お支払いを設定）」を押す
      ③ 表示される Google Cloud のお支払い ↗ でカードを登録し、そのプロジェクトに請求先を紐付ける
      ④ 反映後はキーの貼り替え不要。これまで通り使えて上限だけ上がります。
      料金目安: Gemini API 料金 ↗（標準モデルは安価。課金はご自身のGoogle側で発生し、アプリは無料のままです）

      OCROCRについて

      写真から文字を読み取る機能（DB編集の「写真から読み取り」など）は、端末内で動くTesseract（無料・オフライン可）を使います。明るい場所で文字が大きく写るよう近づけて撮ると精度が上がります。

      有料で高精度にしたい場合（任意・Cloud Vision）
      APIキーの取り方（初回のみ・10分ほど）:
      1. Google Cloud でプロジェクト作成 ↗（Googleアカウントでログイン→名前を入れて「作成」）
      2. お支払い情報を登録 ↗（カード登録。無料枠あり＝月1,000枚まで無料）
      3. Vision API を「有効にする」 ↗
      4. 「認証情報を作成」→「APIキー」 ↗ でキーをコピー
      5. 下の欄に貼り付けて「保存」→「高精度OCRを使う」をON
      料金目安: 月1,000枚まで無料／以降1,000枚あたり約$1.5 ↗
      ONかつキー設定時のみ、車検証スキャンとDB編集の写真読み取りなど全OCRがCloud Visionになります（課金はご自身のGoogle Cloud側）。未設定なら無料Tesseract。

      CLOUDクラウド同期（社内共有）

      ログインすると、車種DB・車両データ（ナンバー/使用者含む）を社内の全端末で自動共有できます。会社ごとに分離され、他社からは見えません。

      新規会社は運営の承認後に有効化。会社に参加は、その会社の代表管理者の承認後に使えます。
