🔧 点検のやり方ガイド

診断で出てくる「導通確認」「電圧測定」などの実際のやり方をまとめました。現場での確認用にどうぞ。
⚠️ 安全第一：測定前にキーOFF・イグニッションの状態を指示どおりに。エアバッグ/SRS系・ハイブリッド高電圧(オレンジ配線)は専用手順が必要です。正式な数値・手順は必ずFAINESや整備要領書で確認してください。本ガイドは一般的な目安です。

① テスター(マルチメーター)の基本

  • 赤リード＝V/Ω/mA 端子、黒リード＝COM 端子に挿す。
  • ダイヤルを測りたい項目へ：V⎓(直流電圧)／Ω(抵抗)／🔊(導通ブザー)。
  • 車の電装は基本 直流(DC)。交流(V〜)ではない。
  • オートレンジでなければ、車はDC20Vレンジを常用。

② 導通(通電)確認のやり方

「線がつながっているか」「スイッチが通電するか」を調べる。

  1. 必ず電源を切る(キーOFF・その回路の電気を止める)。通電したまま抵抗/導通を測らない。
  2. ダイヤルを導通(🔊ブザー)またはΩに合わせる。
  3. 両リードを先に触れ合わせ、ブザーが鳴る/0Ω付近になるか確認(テスターの動作チェック)。
  4. 調べたい線の両端にリードを当てる。ブザーが鳴れば導通あり(つながっている)。鳴らなければ断線の疑い。
  5. スイッチなら、押した時に導通・離すと非導通、が正常か確認。
💡 コネクタを抜いて、線の両端の同じ端子で測るのがコツ。導通ブザーは手軽だが、微妙な抵抗増加は⑤の抵抗測定で数値を見る。

③ 電圧測定のやり方

「その端子に電気が来ているか」を調べる(通電状態で測る)。

  1. ダイヤルをDC電圧(V⎓)に。
  2. 黒リードを確実なボディアース(塗装されていない金属)へ。
  3. 赤リードを測りたい端子へ当てる。
  4. 電源電圧の目安：バッテリー 12.4〜12.9V(停車時)／エンジン始動時 13.5〜14.5V(充電電圧)。
  5. ECU電源ラインなどは、キーONで規定電圧が来ているか、アース側が0Vか確認。

④ 電圧降下テストのやり方

配線や接点の「隠れた抵抗(接触不良・腐食)」を見つける最強の方法。負荷をかけた状態で測る。

  1. ダイヤルをDC電圧に。回路を作動させた状態(電流を流した状態)にする。
  2. 調べたい区間の両端にリードを当てる(例：バッテリー＋端子 と 機器の＋端子)。
  3. 表示が0.2V以下ならほぼ正常。0.3〜0.5V以上は接触不良・腐食・線の細り。
  4. アース側も同様に(機器のボディ と バッテリー−)で測り、0.1V以下が目安。
💡 「電圧は来てるのに動きが悪い」時はこのテスト。導通OKでも電圧降下で不良が出ることが多い。

⑤ 抵抗(Ω)測定のやり方

  1. 必ず電源OFF(通電中は測らない/テスターが壊れる・誤値)。
  2. ダイヤルをΩに。両リードを触れ合わせて0Ω付近を確認。
  3. 部品/配線の両端にリードを当て、数値を読む。
  4. 目安：正常な線＝ほぼ0Ω／断線＝∞(OL表示)／センサ・コイル等は規定値と比較。

⑥ アース(ボディアース)点検

  1. アース線・アースポイントのボルトの緩み・サビ・塗装噛み込みを目視。
  2. 機器のアース端子 と バッテリー−端子 で導通(0Ω付近)を確認。
  3. 作動状態で④の電圧降下(アース側0.1V以下)を確認。
  4. 不良なら端子を磨く/締め直す/アースポイント増設。

⑦ 配線の断線・ショート点検

断線(切れ)

  1. 両端コネクタを抜き、線の両端で導通を測る → 鳴らなければ断線。
  2. ハーネスを軽く動かしwhile測定 → 断続的に切れる(接触不良)を炙り出す。

ショート(短絡)

  1. 天絡(電源とショート)：電源OFFで、その線 と ボディアース の抵抗を測る → 本来∞なのに導通があれば地絡。
  2. 他線とのショート：疑う2本の線の間の抵抗を測る → 導通があれば混線。

⑧ CAN通信ラインの点検

  1. キーOFF・バッテリー等の条件を整えて、CAN-H と CAN-L 間の終端抵抗を測る。
  2. 目安：約60Ω(±6Ω)(2つの120Ω終端抵抗の並列)。
    ・120Ω前後＝終端抵抗が片方だけ/断線の疑い。
    ・∞や大きい値＝配線断線。
    ・0Ω付近＝CAN-HとCAN-Lのショート。
  3. 各CAN線のボディアースへの短絡・電源への短絡も抵抗で確認。
  4. コネクタの接触不良・腐食を目視＋④電圧降下で確認。

⑨ コネクタ・端子の点検

  • 端子の緑青(腐食)・焼け・広がり(勘合不良)を目視。
  • メス端子の保持力(オス端子を挿して緩くないか)。
  • 防水コネクタのシールの劣化・浸水跡。
  • ピンを軽く引いて抜け・かしめ不良を確認。
  • 接点復活剤や端子磨き→ダメなら端子/コネクタ交換。
数値の早見(一般的な目安・要確認)
  • バッテリー電圧(停車)：12.4〜12.9V
  • 充電電圧(始動時)：13.5〜14.5V
  • 電圧降下(＋側区間)：0.2V以下／アース側：0.1V以下
  • 正常な配線の抵抗：ほぼ0Ω／断線：∞(OL)
  • CAN終端抵抗(H-L間)：約60Ω
※ 車種・回路で異なります。正式値は必ずFAINES/整備要領書で確認してください。
